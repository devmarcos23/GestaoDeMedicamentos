Sobre o Projeto de Gestão de Medicamentos


    Bem-vindo ao projeto de Gestão de Medicamentos, uma iniciativa inovadora que criei com base na minha formação no curso de Formação Inicial e Continuada de Desenvolvimento (FICDEV).  Este sistema foi desenvolvido para fins de aprendizagem do funcionamento de sistemas de gestão de medicamentos em casos reais, com o objetivo de otimizar a administração de medicamentos em uma variedade de contextos, incluindo hospitais, clínicas e farmácias.


O curso que deu Origem ao Projeto


    Minha jornada teve início com a inspiradora formação que recebi no curso FICDEV em março de 2023. Este curso notável foi oferecido pela Secretaria de Estado de Ciência, Tecnologia e Inovação do Estado de Mato Grosso (SECITECI) em parceria com o Centro de Inovação, Redes Inteligentes e Soluções Criativas (RISC) da Universidade do Estado de Mato Grosso (UNEMAT), e sua dedicação está em capacitar indivíduos, como eu, com habilidades essenciais em tecnologia. Durante esse programa, adquiri conhecimentos iniciais em uma variedade de tecnologias, que serviram como alicerces fundamentais para a criação deste projeto.


Durante esse curso, que culminou em setembro do mesmo ano, eu adquiri conhecimento nas seguintes tecnologias:


  • BackEnd - Java: A linguagem de programação Java foi a base sólida que permitiu a criação da API para esse sistema de gestão de medicamentos.
  • Banco de dados - PostgreSQL: O curso abordou a administração de bancos de dados, garantindo que os dados fossem armazenados com segurança e eficiência.
  • Criação de páginas dinâmicas - Thymeleaf: A capacidade de criar páginas web dinâmicas usando Thymeleaf melhorou significativamente a usabilidade do sistema.
  • FrontEnd - HTML, JavaScript, CSS: As tecnologias web HTML, JavaScript e CSS permitiram uma projeção de uma interface de usuário atraente e funcional.
  • Design de Interface - Figma: Para criar o design visual do projeto, utilizei o software Figma, que me permitiu desenvolver uma interface elegante e amigável para os usuários.
  • Framework Spring Boot: O conhecimento adquirido sobre o Spring Boot foi fundamental para o desenvolvimento eficiente do sistema.
